# shots_fired
A 2D shooter gun flip game!
## About the Game -
It is 2D fun Physics game based on Godot and GDscript and art work is totaly done on Krita!

## About the Gameplay -
After clicking the play button on menu screen, the the default gun positon is downward with only upward scrolling camera
Player have to manage w/ reflexes to shoot in correct direction and angle to only get upward velcoty as max as possible!
And collect coins and they can collect bullets too!

## Installation
User can play it in browser on Itch.io -> https://basic-bitch-foundation.itch.io/shots-fired and from there itself one can download a runnable acoording to feature this game is especially designed for Phone!
