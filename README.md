# shots_fired
A 2D shooter gun flip game!
## About the Game -
It is 2D fun Physics game based on Godot and GDscript and art work is totaly done on Krita!

## About the Gameplay -
After cliking the play button on menu screen, the the defaul fun positon is dowward with onlt upward scrolling camera
Player have to manage w/ reflexes to shoot in correct direction adn angle to only get upward velcoty as max as possible!
And collect coins and they can collect bullets too!
